## ğŸ“ Customization

### Update Personal Information

1. **Edit HTML Content**: Open `docs/index.html` and update:
   - Name and title in the hero section
   - About me description
   - Work experience entries
   - Skills and percentages
   - Contact information

2. **Update Styles**: Modify `docs/styles/style.css`:
   - Change color scheme in `:root` CSS variables
   - Adjust spacing, fonts, and sizes
   - Customize animations and transitions

3. **Update JavaScript**: Edit `docs/js/main.js`:
   - Modify typing animation titles
   - Adjust animation speeds and delays
   - Customize interactive behaviors

### Change Color Theme

Update the CSS variables in `style.css`:

```css
:root {
  --primary-color: #FFD700;      /* Your primary color */
  --secondary-color: #FFEB3B;     /* Your secondary color */
  --accent-color: #FFC107;        /* Your accent color */
  --dark-bg: #000000;             /* Background color */
  --card-bg: #1a1a1a;             /* Card background */
}
```
